# OpenAI Agents SDK E2B Notebooks

This folder contains Jupyter notebook walkthroughs for the E2B sandbox examples.

Entries:

- `ci_cd_parallel_runner.ipynb`
  Parallel CI-style validation with separate test, lint, and preview sandboxes.

- `data_science_parallel_workbench.ipynb`
  Parallel data-analysis sandboxes that split EDA, cleaning, and dashboard generation.

- `forecast_model_bakeoff.ipynb`
  Notebook walkthrough for the parallel forecast bakeoff example.

- `homepage_prototype_parallel.ipynb`
  Notebook walkthrough for the parallel homepage prototype studio example.

- `homepage_vite_basic_updated.ipynb`
  A notebook walkthrough that iterates on a basic Vite app inside an E2B sandbox using the Agents SDK.

- `parallel_anomaly_triage.ipynb`
  Notebook walkthrough for the parallel anomaly triage example.

- `preview_environment_builder.ipynb`
  A smaller preview-focused example where the agent builds and serves a tiny app from inside the sandbox.

- `sarima_grid_search_parallel.ipynb`
  Notebook walkthrough for the parallel SARIMA grid search example.


- `webpage_designer_example.ipynb`
  A notebook example that uses an E2B sandbox to generate and iterate on a webpage design.
